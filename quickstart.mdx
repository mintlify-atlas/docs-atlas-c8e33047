This page walks through the most common Gamdl commands. Make sure you have [installed Gamdl](/installation) and placed your `cookies.txt` file in your working directory before continuing.

## Basic syntax

```bash
gamdl [OPTIONS] URLS...
```

You can pass one or more Apple Music URLs. Wrap each URL in quotes to avoid your shell interpreting special characters.

## Common commands

<Steps>
  <Step title="Check the version">
    Confirm Gamdl is installed and check its version:

    ```bash
    gamdl --version
    ```
  </Step>

  <Step title="View all options">
    Print the full help output with all available flags:

    ```bash
    gamdl --help
    ```
  </Step>

  <Step title="Download a single song">
    Pass a song URL (one that includes `?i=` — the track identifier in an album URL):

    ```bash
    gamdl "https://music.apple.com/us/album/never-gonna-give-you-up-2022-remaster/1624945511?i=1624945512"
    ```

    The file is saved to `./Apple Music/` by default, organized as `{album_artist}/{album}/{track:02d} {title}.m4a`.
  </Step>

  <Step title="Download a full album">
    Pass an album URL (without a `?i=` parameter) to download every track:

    ```bash
    gamdl "https://music.apple.com/us/album/whenever-you-need-somebody-2022-remaster/1624945511"
    ```
  </Step>

  <Step title="Download from an artist page">
    Pass an artist URL to open an interactive selection prompt:

    ```bash
    gamdl "https://music.apple.com/us/artist/rick-astley/669771"
    ```

    Gamdl fetches the artist's catalog and presents a list. Use the following controls to make your selection:

    | Key | Action |
    | --- | --- |
    | Arrow keys | Move the cursor up and down |
    | Space | Toggle the item under the cursor |
    | Ctrl + A | Select all items |
    | Enter | Confirm selection and start downloading |

    To skip the prompt and download a specific category automatically, use `--artist-auto-select`:

    ```bash
    gamdl --artist-auto-select main-albums "https://music.apple.com/us/artist/rick-astley/669771"
    ```

    Valid values for `--artist-auto-select` are: `main-albums`, `compilation-albums`, `live-albums`, `singles-eps`, `all-albums`, `top-songs`, `music-videos`.
  </Step>

  <Step title="Specify a cookies file">
    If your cookies file is not named `cookies.txt` or is not in the working directory, use `--cookies-path`:

    ```bash
    gamdl --cookies-path /path/to/cookies.txt "https://music.apple.com/us/album/never-gonna-give-you-up-2022-remaster/1624945511?i=1624945512"
    ```
  </Step>

  <Step title="Change the output directory">
    Use `--output-path` to save downloads to a specific folder:

    ```bash
    gamdl --output-path /music "https://music.apple.com/us/album/whenever-you-need-somebody-2022-remaster/1624945511"
    ```
  </Step>
</Steps>

## Next steps

- Set persistent defaults by editing the config file at `~/.gamdl/config.ini` (Linux) or `%USERPROFILE%\.gamdl\config.ini` (Windows). The file is created automatically on the first run.
- Use the `--song-codec-priority` flag to request a different audio codec, or enable the Wrapper to access lossless ALAC and Dolby Atmos.
- Pass `--synced-lyrics-format srt` or `--synced-lyrics-format ttml` to change the lyrics format from the default LRC.
- Download multiple URLs in a single command by separating them with spaces, or use `--read-urls-as-txt` (`-r`) to read URLs from a plain text file.
